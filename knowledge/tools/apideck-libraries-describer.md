Analyze codebases using AI - generate architectural overviews, documentation, explanations, bug reports and more.

## Key Features
- Scan all files in a directory and use Google's Gemini AI to generate analysis
- Generate markdown architectural overviews
- Summarize codebases in multiple formats
- Analyze code patterns and structures
- Document codebase for new developers
- Find potential bugs or issues
- Generate test ideas
- Respect .gitignore rules when scanning files
- Support for custom analysis prompts
- Save output to markdown files
- Exclude specific file patterns (tests, minified code, etc.)

## Use Cases
Perfect for developers who need to quickly understand unfamiliar codebases, document legacy systems, analyze code patterns, find potential issues, or generate comprehensive architectural documentation. Ideal for onboarding new developers, code reviews, and maintaining code quality across large projects.
