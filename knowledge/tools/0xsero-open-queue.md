A simple, open-source queue management system. An open-source queue management system designed for events, appointments, or any scenario where you need to manage people in line.

## Key Features
- Queue management with basic FIFO (First-In-First-Out) operations
- Waitlist management - add, remove, and manage people on the waitlist
- Priority queues support for priority-based queueing
- Real-time updates using Socket.IO for instant queue status updates
- REST API for integration with other systems
- Web interface for easy queue management
- Built with TypeScript, Node.js, PostgreSQL, and Socket.IO
- Self-hosted option available
- Scalable architecture supporting hundreds of queues

## Use Cases
Perfect for managing event registration queues, service appointments, customer support systems, healthcare patient tracking, or any scenario where you need to efficiently manage people or requests in line with real-time status updates.
