A web based dashboard for your SST apps.

## Key Features
- Web-based dashboard for managing SST applications with your team
- Accessible at console.sst.dev
- Built with TypeScript for type safety
- AWS integration with read-only permissions to everything in the AWS account
- Team collaboration features
- Real-time monitoring and management of SST applications

## Use Cases
Perfect for teams developing SST (Serverless Stack) applications who need a centralized dashboard for monitoring and managing their serverless infrastructure. Ideal for collaboration, visibility into deployed applications, and AWS account management with proper security boundaries.
