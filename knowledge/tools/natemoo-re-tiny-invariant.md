Tiny-invariant is a minimal invariant/assertion library for JavaScript/TypeScript applications. It provides a lightweight way to add runtime checks and assertions to your code, helping catch errors early in development and ensuring code contracts are maintained.

## Key Features

- **Lightweight** - Minimal footprint with no external dependencies
- **TypeScript Support** - Full TypeScript type definitions and type safety
- **Runtime Checks** - Catch errors early with invariant assertions
- **Customizable Messages** - Support for custom error messages
- **Production Safe** - Optimized for production environments with proper tree-shaking
- **Simple API** - Easy to use with straightforward syntax
- **Bundle Friendly** - Works well with modern bundlers
- **Cross-Platform** - Works in Node.js, browsers, and other JavaScript environments

## Use Cases

- Adding runtime type checking and validation
- Ensuring function preconditions and postconditions
- Debugging and development error checking
- API contract validation
- Data integrity checks in applications
- Testing environments with additional assertions
- Production code with invariant requirements
