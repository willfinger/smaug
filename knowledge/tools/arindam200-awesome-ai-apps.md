Awesome AI Apps is a comprehensive collection of 70+ practical examples, tutorials, and recipes for building powerful LLM-powered applications. This curated repository serves as an essential guide for developers working with various AI frameworks, from simple chatbots to advanced AI agents and production-ready workflows.

## Key Features

- **70+ Practical Examples**: Complete implementations showcasing RAG, agents, workflows, and other AI use cases
- **Categorized Projects**: Well-organized into 6 main categories (Starter Agents, Simple Agents, MCP Agents, Memory Agents, RAG Applications, Advanced Agents)
- **AWS Strands Course**: Comprehensive 8-lesson course on building AI agents with AWS Strands SDK
- **Framework Diversity**: Examples across multiple frameworks including OpenAI SDK, LlamaIndex, CrewAI, LangChain, Agno, and more
- **Production-Ready**: Advanced agents with multi-stage pipelines, complex workflows, and enterprise features
- **Memory Integration**: Projects featuring advanced memory capabilities with Memori v3
- **MCP Integration**: Examples using Model Context Protocol for external tool integration
- **Video Tutorials**: Comprehensive video resources alongside code examples
- **Active Community**: Regularly updated with new projects and contributions from the community
- **Rapid Growth**: Crossed 8k stars milestone with significant community engagement

## Use Cases

- Learning AI agent development from beginner to advanced levels
- Finding reference implementations for specific AI use cases
- Understanding different AI frameworks and their applications
