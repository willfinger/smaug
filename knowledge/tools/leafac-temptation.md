Temptation is a PHP library designed to painlessly create and manage temporary files and directories. It provides a robust solution for handling temporary resources that automatically clean up after themselves, preventing resource leaks and ensuring proper cleanup even in error scenarios.

## Key Features

- **Automatic Cleanup** - Files and directories are automatically cleaned up
- **Exception Safety** - Ensures cleanup even when exceptions occur
- **Flexible API** - Simple and intuitive API for common operations
- **Multiple Formats** - Support for various file formats (text, binary, CSV, JSON)
- **Directory Support** - Create and manage temporary directories
- **File Permissions** - Configurable file permissions
- **Path Resolution** - Easy resolution of temporary file paths
- **Atomic Operations** - Support for atomic file operations
- **Error Handling** - Comprehensive error handling and reporting
- **Cross-Platform** - Works across different operating systems

## Use Cases

- **Testing Frameworks** - Create temporary files for unit tests
- **File Processing** - Handle temporary files during data processing
- **Caching** - Implement temporary caching mechanisms
- **Import/Export** - Manage temporary files during data import/export
- **Report Generation** - Create temporary report files
- **Image Processing** - Handle temporary image files
- **API Responses** - Generate temporary response files
- **Database Operations** - Manage temporary database dumps
- **Build Processes** - Handle temporary files during build processes
- **Development Tools** - Create temporary files for development workflows
