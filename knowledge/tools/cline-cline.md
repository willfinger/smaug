Cline is an AI assistant that can use your CLI and Editor, offering an autonomous coding experience with human-in-the-loop safety. Thanks to Claude Sonnet's agentic coding capabilities, Cline can handle complex software development tasks step-by-step with your permission.

## Key Features

- **File Creation & Editing**: Create and edit files directly in your editor with diff views and automatic error monitoring (missing imports, syntax errors)
- **Terminal Command Execution**: Execute commands directly in your terminal with real-time output monitoring
- **Browser Integration**: Launch browsers, click elements, type text, and scroll to test applications and fix visual bugs
- **Custom Tool Creation**: Extend capabilities using the Model Context Protocol (MCP) to create and install custom tools
- **Multiple API Support**: Works with OpenRouter, Anthropic, OpenAI, Google Gemini, AWS Bedrock, Azure, GCP Vertex, Cerebras, Groq, and local models
- **Workspace Management**: Take snapshots, compare changes, and restore previous states
- **Context Enhancement**: Add URLs, workspace problems, files, and folders to enhance AI understanding

## Use Cases

- Building complex applications from mockups or screenshots
- Debugging and fixing code with visual feedback
- Running end-to-end tests on web applications
- Managing large codebases and projects
- Creating custom tools for specific workflows
- Learning and exploring new coding techniques
