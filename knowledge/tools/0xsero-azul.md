A beautiful terminal web browser with AI-powered chat, built in Rust. Browse the world in the comfort of your terminal.

## Key Features
- Full TUI web browsing with keyboard navigation
- AI chat panel with tool-calling support (navigate, scroll, follow links)
- Multi-engine search (DuckDuckGo, Wikipedia, arXiv, PubMed, Google Scholar, OpenLibrary)
- JavaScript rendering via headless Chrome
- Bookmarks and history
- Cosmic dark theme
- Tab support
- RAG integration (optional)
- Keyboard-driven interface with comprehensive keybindings
- Support for multiple AI providers (OpenRouter, OpenAI, Ollama, custom endpoints)

## Use Cases
Perfect for developers who want to browse the web without leaving their terminal, researchers who need quick access to multiple search engines, and anyone who wants to combine web browsing with AI assistance in a single, distraction-free environment.
