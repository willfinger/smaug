Squeleton is a Python command-line tool to generate project skeletons from a template. It allows developers to quickly scaffold new projects with standardized structures, reducing the time spent on boilerplate code setup and focusing on building the actual application features.

## Key Features

- **Project Template Generation**: Create complete project structures from templates
- **Multiple Framework Support**: Templates for popular frameworks and languages
- **Custom Templates**: Support for user-defined project templates
- **Configuration Management**: Easy configuration of project-specific settings
- **Dependency Management**: Automatically install project dependencies
- **Git Integration**: Initialize git repositories with .gitignore files
- **VS Code Support**: Generate VS Code workspace configurations
- **Environment Setup**: Create environment files and configuration
- **Documentation Generation**: Auto-generate README and documentation files
- **Plugin Architecture**: Extensible system for custom generators

## Use Cases

- Quickly starting new projects with established patterns
- Ensuring consistent project structures across teams
- Reducing boilerplate code creation time
- Setting up microservices with standardized patterns
- Creating new API endpoints with consistent structure
- Generating frontend React/Vue/Angular projects
- Setting up Python projects with proper structure
- Creating boilerplate for testing frameworks
- Generating documentation alongside project creation
- Onboarding new developers with consistent project layouts
