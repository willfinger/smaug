Universal skills loader for AI coding agents - npm i -g openskills. OpenSkills brings Anthropic's skills system to every AI coding agent, allowing you to use Claude Code skills with Cursor, Windsurf, Aider, Codex, and any agent that can read AGENTS.md.

## Key Features

- **Exact Claude Code compatibility** - Same prompt format, same marketplace, same folder structure
- **Universal agent support** - Works with Claude Code, Cursor, Windsurf, Aider, Codex, and more
- **Progressive disclosure** - Load skills only when needed (keeps context clean)
- **Repo-friendly** - Skills live in your project and can be versioned
- **Private friendly** - Install from local paths or private git repos
- **One CLI for all agents** - Unified interface across multiple AI coding tools

## Use Cases

- Adding Claude Code skills to non-Claude Code environments
- Sharing skills across different AI coding agents in your workflow
- Managing and versioning skills within your Git repositories
- Using private or local skill repositories
- Multi-agent setups where you want consistent skill access
- Progressive loading of skills to maintain clean context windows
