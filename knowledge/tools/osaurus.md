Native macOS LLM server with MCP support. Osaurus is an all-in-one platform for running language models on Apple Silicon with OpenAI and Anthropic compatible APIs, MCP server capabilities, and a built-in plugin ecosystem.

The tool enables Claude and other AI agents to see your screen, click, type, and navigate - perfect for demonstrating what AI agents look like on modern macOS systems. It combines local MLX inference with support for remote providers (Anthropic, OpenAI, OpenRouter, etc.).

## Key Features

- MLX Runtime for optimized Apple Silicon inference
- OpenAI, Anthropic, and Ollama compatible APIs
- MCP Server for integration with Cursor, Claude Desktop
- Multi-window chat with custom personas
- Voice input with WhisperKit transcription
- Plugin system for extensibility
- Menu bar chat overlay
- Support for Apple Foundation Models (macOS 26+)
