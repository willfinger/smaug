Integrates ChatGPT and local LLMs into Obsidian

## Key Features

- Markdown formatting for ChatGPT conversations
- Export ChatGPT chats to structured markdown
- Preserve code blocks and formatting
- Organize conversation history
- Documentation generation from AI chats


## Use Case

Developers and researchers use this to archive and organize their AI conversations for documentation purposes.
