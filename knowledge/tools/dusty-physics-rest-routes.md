REST Routes - A lightweight routing library for building REST APIs with ease.

## Key Features
- Lightweight and fast routing performance
- Easy-to-use API for defining REST endpoints
- Built-in middleware support for request processing
- Full TypeScript support with type safety
- Automatic route generation and documentation
- Flexible configuration options for customization

## Use Cases
- Building RESTful APIs quickly and efficiently
- Microservices architecture development
- API prototyping and rapid development
- Projects requiring TypeScript type safety
- Applications needing middleware support
- Documentation generation for API endpoints
