- Drop transition effect
- Auto rotation with paging
- Text effect
- Text transition
- Fade-in/out transition/animation
- Beautiful preview samples (Apple TV, Final Cut Camera, Find My, Journal styles)
- Support for iOS 18+
- Built with Swift 6.0

## Use Cases
Perfect for creating modern, animated splash screens for iOS 18+ applications with smooth transitions and auto-rotating content. Suitable for apps that want to showcase Apple's latest design patterns.
