Rizz (ریز) is a tiny, multi-platform, and minimal game development framework written in the C programming language. Created by Sepehr Taghdisian, this lightweight framework provides developers with a no-frills solution for creating games and applications with minimal dependencies and maximum cross-platform compatibility. Inspired by libraries like The Machinery and sokol, Rizz is designed for developers who prefer simplicity and efficiency over bloated game engines.

## Key Features

- **Minimalist Design** - Small and lightweight framework with minimal dependencies
- **Cross-Platform** - Supports Linux, macOS, and Windows with consistent behavior
- **Pure C Implementation** - Written entirely in C for maximum performance and portability
- **Lightweight** - Minimal memory footprint and fast execution
- **No External Dependencies** - Self-contained framework without requiring complex build systems
- **Flexible Architecture** - Designed for simplicity and easy integration
- **Game Engine Features** - Basic game development capabilities without the bloat
- **Example Projects** - Includes Space Invaders remake demonstrating framework usage

## Use Cases

- **Indie Game Development** - Create small to medium games with minimal overhead
- **Rapid Prototyping** - Quickly prototype game ideas without complex setup
- **Learning Game Development** - Study game programming fundamentals in C
- **Multi-platform Games** - Build games that run on multiple desktop platforms
- **Modular Game Components** - Create reusable game systems and components
- **Performance-Critical Applications** - Where C performance is required
- **Educational Projects** - Teaching game programming concepts
- **Lightweight Applications** - Games and apps where resource usage matters
